# **App Name**: Mnemonix

## Core Features:

- Smart Deck Generator: Uses AI as a tool to generate tailored flashcard decks based on any specific subject or uploaded text summary.
- Focus Mode Player: A minimal, distraction-free interface for studying cards with interactive 3D flip animations.
- Curated Library: An organized dashboard to manage and categorize different learning modules for quick access.
- Adaptive Mastery Tracking: A visual system to mark cards as mastered or requiring review, helping to track learning progress over time.

## Style Guidelines:

- A deep 'Focus' dark theme: Primary Electric Indigo (#3666E2), deeply desaturated Obsidian background (#101318), and an Arctic Cyan accent (#26B6D9).
- Font pairing: 'Space Grotesk' (Sans-serif) for high-impact headers to convey a tech-forward feel, paired with 'Inter' (Sans-serif) for body text to ensure maximum readability during long study sessions.
- Clean, modular grid-based layout inspired by luxury design portfolios, with generous whitespace and a center-focused study card orientation.
- Ultra-thin line icons with a uniform stroke weight to maintain a professional, sophisticated aesthetic across the dashboard.
- Smooth, physics-based CSS 3D transitions for flipping flashcards and subtle stagger-entry animations for deck items.
